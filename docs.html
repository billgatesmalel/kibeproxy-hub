Using Your Proxy
Configure your proxy in different environments
1
Get Your Credentials
Dashboard → Active tab → copy Host, Port, Username and click Show for Password.
2
Python (Requests)
🐍 Python
import requests proxy = { 'http': 'http://username:password@host:port', 'https': 'http://username:password@host:port' } response = requests.get('https://httpbin.org/ip', proxies=proxy) print(response.json()) # Shows your proxy IP
3
Node.js (Axios)
🟨 Node.js
const axios = require('axios'); const { HttpsProxyAgent } = require('https-proxy-agent'); const agent = new HttpsProxyAgent( 'http://username:password@host:port' ); const res = await axios.get('https://httpbin.org/ip', { httpsAgent: agent }); console.log(res.data); // Shows your proxy IP
4
