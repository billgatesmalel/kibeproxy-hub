SMTP Setup Guide

After purchasing an email account from KibeProxy Hub, you can use it to send emails via SMTP. Here's how to configure it:

1
Get Your Email Credentials

Go to your Dashboard → click the Emails tab → click Show to reveal your email and password.

2
Configure SMTP Settings

Use these settings in your email client or app:

SMTP Hostsmtp.gmail.com
SMTP Port587 (TLS) or 465 (SSL)
Usernameyour purchased email
Passwordyour email password
EncryptionTLS / STARTTLS
3
Using with Python

Example code to send an email using Python:

import smtplib
from email.mime.text import MIMEText

smtp_host = "smtp.gmail.com"
smtp_port = 587
email     = "your@email.com"
password  = "yourpassword"

msg = MIMEText("Hello from KibeProxy!")
msg['Subject'] = "Test Email"
msg['From']    = email
msg['To']      = "recipient@email.com"

with smtplib.SMTP(smtp_host, smtp_port) as s:
    s.starttls()
    s.login(email, password)
    s.send_message(msg)
    print("Email sent!")
4
Using with Node.js (Nodemailer)
const nodemailer = require('nodemailer');

const transporter = nodemailer.createTransport({
  host: 'smtp.gmail.com',
  port: 587,
  auth: {
    user: 'your@email.com',
    pass: 'yourpassword'
  }
});

await transporter.sendMail({
  from: 'your@email.com',
  to:   'recipient@email.com',
  subject: 'Test Email',
  text: 'Hello from KibeProxy Hub!'
});
